# Ace of Fades Website

Welcome to the **Ace of Fades Website** repository! This is the official codebase for the Ace of Fades Luxury Men's Salon. The application is built with a modern tech stack to provide a fast, responsive, and seamless experience for users.

## 🚀 Tech Stack

### Frontend (Client)
- **Framework**: React 19 + Vite
- **Styling**: Sass, Framer Motion for animations
- **Routing**: React Router
- **Icons**: Lucide React
- **Language**: TypeScript

### Backend (Server)
- **Framework**: Express.js (Node.js)
- **Database ORM**: Prisma
- **Security & Middleware**: Helmet, CORS, Compression, Express Rate Limit
- **Language**: TypeScript

## 📁 Project Structure

- `/client` - Contains the React front-end code.
- `/server` - Contains the Node.js/Express API and Prisma database schema.

## 🛠️ Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/DEViLxPLAYS/AceofFadesWebsite.git
   cd AceofFadesWebsite
   ```

2. **Install dependencies for the server:**
   ```bash
   cd server
   npm install
   ```

3. **Install dependencies for the client:**
   ```bash
   cd ../client
   npm install
   ```

### Running the Application Locally

You will need two separate terminal windows/tabs to run the client and the server.

**Terminal 1 (Backend - Server):**
```bash
cd server
npm run dev
```

**Terminal 2 (Frontend - Client):**
```bash
cd client
npm run dev
```
